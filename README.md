# Attract Lib

Premiere Pro UXP extension panel. Built with Bolt UXP (React + TypeScript + Vite).

## Prerequisites

- Adobe Premiere Pro 2024+ (v24.0 or later)
- UXP Developer Tools (UDT) v2.2.1+
- Node.js 18+
- npm

## Getting Started

```bash
npm install
npm run build
npm run dev
```

## Distribution

```bash
npm run ccx
```

Produces a `.ccx` file in `dist/`. Double-click to install via Creative Cloud Desktop.
